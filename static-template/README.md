# Marina di Concaverde - Sito Web Ufficiale

Benvenuti nel repository del sito web di **Marina di Concaverde**, un incantevole campeggio situato nella splendida Sardegna, precisamente a Santa Teresa Gallura. Questo progetto rappresenta una rifattorizzazione completa del sito web originale, con l'obiettivo di offrire un'esperienza utente moderna, intuitiva e visivamente accattivante.

## 🌿 Descrizione

Marina di Concaverde è un caratteristico borgo che sorge su un promontorio nell'estremità settentrionale della Sardegna, affacciato sulle Bocche di Bonifacio. Il campeggio offre un punto ideale per partire alla scoperta dell'arcipelago e delle sue incantevoli spiagge, protette e accessibili.

Questo sito web è stato completamente rinnovato per riflettere la bellezza naturale e la serenità del luogo, presentando:

- **Alloggi unici**: Piazzole in pineta, bungalow lago e chalet vista lago
- **Spiaggia privata**: Accesso diretto a una spiaggia incontaminata
- **Servizi completi**: Tutto il necessario per un soggiorno rilassante
- **Territorio mozzafiato**: Immerso nella natura sarda più autentica

## ✨ Caratteristiche

- **Design Responsivo**: Ottimizzato per desktop, tablet e dispositivi mobili
- **Interfaccia Moderna**: Utilizzo di Tailwind CSS per un aspetto pulito e professionale
- **Galleria Fotografica**: Immagini ad alta risoluzione che catturano l'essenza del luogo
- **Navigazione Intuitiva**: Menu chiaro e sezioni ben organizzate
- **Animazioni Sottili**: Transizioni fluide per migliorare l'esperienza utente
- **Contenuto Localizzato**: Testi in italiano, focalizzati sul mercato turistico sardo

## 🚀 Come Utilizzare

Il sito è completamente statico e non richiede installazioni particolari:

1. **Clona il repository**:

   ```bash
   git clone https://github.com/orso081980/marinadiconcaverde.git
   cd marinadiconcaverde
   ```

2. **Apri nel browser**:
   - Apri il file `index.html` direttamente nel tuo browser web
   - Oppure utilizza un server locale per un'esperienza ottimale:

     ```bash
     # Con Python (se installato)
     python -m http.server 8000

     # O con Node.js
     npx live-server
     ```

3. **Naviga il sito**: Esplora le sezioni dedicate agli alloggi, alla spiaggia, ai servizi e al territorio

## 🛠️ Tecnologie Utilizzate

- **HTML5**: Struttura semantica e accessibile
- **CSS3**: Stili personalizzati con Tailwind CSS
- **JavaScript**: Interattività minima per menu mobile e animazioni
- **Tailwind CSS**: Framework utility-first per un design rapido e consistente
- **Google Fonts**: Tipografia elegante (Cormorant Garamond e DM Sans)
